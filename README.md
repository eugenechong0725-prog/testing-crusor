# ProWeb Studio — Lead Collection Landing Page

A client-ready landing page for collecting customer inquiries. Built with plain HTML, CSS, and JavaScript.

## Features

- Inquiry form: name, phone, service, budget, notes
- Form validation (required fields, phone must have at least 8 digits)
- Success message after submit (no page redirect)
- Lead List with delete support
- Data persisted in browser localStorage
- Responsive layout for mobile and desktop

## Files

| File | Description |
|------|-------------|
| `index.html` | Page structure and form |
| `styles.css` | Layout and styling |
| `app.js` | Validation, localStorage, lead list logic |

## Run locally

```bash
python3 -m http.server 8080
```

Open http://localhost:8080
